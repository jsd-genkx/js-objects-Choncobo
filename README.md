# JavaScript Objects

## Object Syntax

```js
Syntax:

const objectName = {
  key: value,
  methodName() {
    console.log(this.key);
  }
};

```

## Optional Chaining & Nullish Coalescing Operator

```js
Syntax: obj?.nestedKey;

obj.key ?? defaultValue;
```

## Access Object Properties

- Dot Notation

```js
Syntax: obj.key;
```

- Bracket Notation

```js
Syntax: obj["key"];
```
